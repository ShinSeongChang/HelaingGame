1. 각 기획자들 담당하시는 시스템 기획서 완료 / 레벨 디자인 (페이퍼 웍)
	* 문서 완료 시 관련 담당자와 바로 이야기 한 후 일 진행 하실 수 있게 해주시면 됩니다.

	* 해당 문서 완료 후 "넷마블 기획방"에 추가로 올려주세요
		- 부족한 부분은 말 또는 간단하게 정리하여서 보내드리고 다음 할 일 해주시면 감사하겠습니다
		- 아침에는 갠톡으로 문서 보내라고 하지 않으셨나요?
			ㄴ 제가 확인을 못하면 진행 못할 거 같아, 완성이 되시면 바로 해당 담당자랑 이야기하면서 보완해주시면 될 거 같습니다

		### 문서 작성 꼭 주의 사항 ###
	1. 버전 관리 해주세요 / [ 날짜, 바뀐 내용, 바꾼 사람 ]
	2. 문서에 대한 양식을 최대한 맞춰 주세요
	3. 참고할만한 에셋 추가해주기
		* 해당 사항은 문서를 완료 후 프로그래머팀이 진행이 되고, 기획서 쓸게 없을 때 작업해주세요
	
	
2. 담당 프로그래머분들에게 제작 기간 물어보고 + 지라에 업데이트 하기

	* 구현을 하시는데 대강 어느 정도의 기간이 나올 거 같은지 지라에 업데이트 해주세요
		- 해당 부분은 금요일에 와서 일정 확인 후에 다시 조율 하겠습니다

3. 이슈 + 논의가 필요한 부분은 "고지완 갠톡"에 따로 올려주시면 확인되는대로 코멘트 남기겠습니다
	
4. 목요일 전체 일정 요약

	1. 담당 시스템 기획서 작성 

	2. 담당 시스템 관련 플밍분과 대화
		2-1 이슈가 있을 시 지라에 작성

	3. 플밍분이 문제 없이 진행 될 거 할 시
		3-1 일정 작성
		3-2 해당 기획서 참고 에셋 추가

	4.레벨 디자인 기획서 작성
		4-1 맵 플레이 컨셉
		4-2 해당 맵의 특징 (어떠한게 재밌는지)
		4-3 맵에 존재하는 오브젝트
		4-4 해당 맵에 어떠한 경험을 느끼기 위해 어떤 의도로 했는지
		4-5 페이퍼 웍
		4-6 페이퍼 웍을 토대로 맵 FLOW 작성
		
5. 나폴리탄 괴담

6번 문항은 사실 없는 문장입니다.
